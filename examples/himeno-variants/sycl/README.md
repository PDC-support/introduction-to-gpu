# How to use AdaptiveCpp (hipSYCL)

## On LUMI

Use hipSYCL module from CSC:
```
module use /appl/local/csc/modulefiles
module load hipsycl/0.9.4
```

and compile with
```
make
```

## On Dardel

Load a hipSYCL module:
```
ml PDC/22.06
ml hipsycl/0.9.4-cpeGNU-22.06
```

and compile with
```
make
```
